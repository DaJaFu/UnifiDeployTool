Write a program that connects to a Unifi Gateway Max or Unifi Gateway Ultra that is using factory default settings using default login info, but prompts a user for login information if that initially fails. From there, this program will:

    1. Configure 3 VLANs, using configuration files contained within the 
       cwd (generate this file). By default, I would like the VLAN config to 
       be as secure as possible.
    2. Upgrade the firmware of the Gateway. (Store latest version of device firmware in program if necessary for offline deployment.)
    3. Check for any unifi devices connected to the gateway, adopt them, and determine the device type + configure them. 
    4. Apply VLAN settings to necessary ports in a sequence that will prevent any adoption loops or traffic tagging issues from arising.

Once everything is configured, grab any device info, and create documentation, inventory sheets.
